### Этот проект — учебная работа про поиск подозрительных пассажиров для авиакомпании Yoyoflot.
Мы собираем данные из разных источников, приводим их в порядок, объединяем в одну таблицу и пытаемся понять, кто из путешественников ведёт себя странно.


-> очистка и объединение сырых файлов в один итоговый CSV-файл

-> расчёт активности пассажиров и их «подозрительности»

-> простая консольная система поиска по ФИО/дате

-> просмотр подробной информации по выбранному человеку

-> построение HTML-карты его перелётов
