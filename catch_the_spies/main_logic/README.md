## В проекте три файла:

main.py – для запуска, возможность вывода топа подозрительных пассажиров и поиск по именам
analysis_core.py – вся логика обработки данных и считает итоговый «риск-балл»
map_visuals.py – генерация html-карты маршрутов пассажира

Перед запуском нужно положить файл merged_all_detailed.csv в папку data, которая находится на уровень выше (../data/... относительно main.py).
